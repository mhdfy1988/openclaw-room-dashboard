# v0.1.0

Initial public release of the OpenClaw Room Dashboard.

## Highlights

- Pixel-room dashboard for OpenClaw multi-agent status
- Room scene with agent focus mode and detail panel
- Gateway overview, recent events, current connections, and alert center
- OpenClaw settings dialog with local file config support
- Remote gateway degraded-state detection for `status / health / system-presence`
- Background start / stop / status scripts for local development
- Backend and frontend tests plus smoke-test workflow

## Validation

- `npm run test --workspace backend`
- `npm run test --workspace frontend`
- `npm run build --workspace frontend`

## Notes

- Local config files and runtime logs are ignored by Git
- Remote plain `ws://` gateways may only provide partial data until secure access and device pairing are completed
